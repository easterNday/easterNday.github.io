Arch Linux 简明安装指南 #

本文主要介绍最简单的 Arch Linux 脚本安装方法而并非逐步手动安装方法，更多详情请参考archinstall。

如需要自行手动安装，请参考 archlinux 简明指南 和 Arch Linux Studio 中对应部分的教程。

Arch Install 简介 #

archinstall 是一个用于安装 Arch Linux 的帮助库。它和其它的预配置安装程序一起打包，例如“向导”安装程序。

安装方法 #

从Arch官网下载地址选择离你最近的镜像站（例如清华大学 Arch 镜像），下载对应平台架构的镜像文件。

随后利用 balena-etcher 、 rufus 等烧录软件将镜像文件烧录到U盘中。

连接U盘并将电脑重启到对应的 Live OS 中，并进行网络连接(此部分可以参考连接网络)。

之后再终端中运行指令：

bash
archinstall

对应菜单信息列举如下：

  • 配置区域
  • 选择镜像
  • 分区磁盘
  • 格式化分区
  • 启用磁盘加密（可选）
  • 设置计算机名
  • 设置 root 密码
  • 安装引导加载程序

在菜单中进行相关的设置之后即可完成安装。安装完成后会出现一条绿色文本，表示安装完成，可以安全重启。这也是用于重启的命令。

如果您是需要在 Windows 下启用对应的子系统，请参考WSL部分
