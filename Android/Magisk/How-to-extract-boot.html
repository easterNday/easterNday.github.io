# 准备工作

# 下载手机系统包

首先你需要拥有你自己手机机型对应的刷机包，小米Rom下载可参考小米ROM相关，其余机型请暂时麻烦自己查找。

# payload-dumper-go 下载

payload-dumper-goopen in new window 是一款安卓OTA负载一键提取软件。

对应 macOS , Windows 以及 Linux 预构建文件下载链接：Githubopen in new window

# 提取boot.img

# 解压系统包

将之前下载的刷机包解压缩并找到payload.bin文件，将其复制到 payload-dumper-go 的文件夹下。

# 运行程序

对于 Windows 用户，将 payload.bin 拖到目录下的 payload-dumper-go.exe 即可自动开始运行。

# 补充资料

# 注意

  • 出于性能原因，强烈建议使用SSD，HDD可能是一个瓶颈。
  • 不支持增量OTA（增量）负载。
