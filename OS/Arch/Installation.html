Arch Linux 简明安装指南 ​

 自动安装 ​

1. 使用 archinstall 安装 ​

从Arch 官网下载地址选择离你最近的镜像站（例如清华大学 Arch 镜像），下载对应平台架构的镜像文件。

随后利用 balena-etcher 、 rufus 等烧录软件将镜像文件烧录到 U 盘中。

连接 U 盘并将电脑重启到对应的 Live OS 中，并进行网络连接(此部分可以参考连接网络)。

之后再终端中运行指令：

bash
archinstall

对应菜单信息列举如下：

  • 配置区域
  • 选择镜像
  • 分区磁盘
  • 格式化分区
  • 启用磁盘加密（可选）
  • 设置计算机名
  • 设置 root 密码
  • 安装引导加载程序

在菜单中进行相关的设置之后即可完成安装。安装完成后会出现一条绿色文本，表示安装完成，可以安全重启。这也是用于重启的命令。

2. 使用具有图形安装界面的第三方 Arch Linux 改版 ​

如果您只是想获得一个开箱即用的 Arch Linux 而不用经过繁琐的安装过程（其实也没有很繁琐），那么您可以尝试一下如下几个发行版：

  • Manjaro: 目前使用人数最多的第三方发行版本
  • Archcraft: 基于 ArchLinux 的第三方版本，个人目前在用，部分版本需要收费
  • ArchLinux GUI: 这是一个纯净的 Arch Linux,只预装了桌面环境

 手动安装 ​

手动使用命令行安装请阅读官方指南。

如果您想使用 LVM 将多个磁盘模拟为一整个磁盘的话，可以参考官方文档来进行分区划分，或者使用 GParted 进行划分后再自行完后扩容操作。
挂载部分指令基本不变（注意分区名字，一般前缀为/dev/mapper/***）
安装时记得修改 /etc/mkinitcpio.conf 并在 HOOKS 部分在 block 和 filesystems之间添加lvm2 后执行 mkinitcpio -p linux 生成配置文件(linux 部分为你使用的内核)
更多操作请查看参考中的文章

如果您觉得官方文档比较难理解，也可以参考 archlinux 简明指南 和 Arch Linux Studio 指南 中对应部分的教程。

如果您是需要在 Windows 下启用对应的子系统，请参考WSL 部分
